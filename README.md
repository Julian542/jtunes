# JTunes
## Descripcion:
Simplemente una store de canciones al estilo ITunes hecha para estudiar Spring
## Tecnologias utilizadas:
* Apache Netbeans
* Maven
* Java 8
* Spring Boot 
* Thymeleaf
* HTML5
* CSS
* Bootstrap
* MySQL
## Dependencias del proyecto SpringBoot:
* Spring Web
* Spring Data JPA 
* Thymeleaf
* Spring Boot DevTools
* HyperSQL Database (h2)
* MySQL Driver
